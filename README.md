## Follow the steps to Run the project 
1. Open terminal in project folder
2. Run `npm install` to install required node modules
3. Run `npm start` to run react frontend
4. Run `npm run server` to run json-server(mock backend) which runs on [http://localhost:5000/tasks](http://localhost:5000/tasks)
5. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
